Not another memory tool.

Memory tools store data for one agent on one machine. Relay is a protocol that lets humans and AI agents coordinate context across teams, sessions, and vendors — like email for the agentic era.

◎ Multi-Agent Coordination

Every agent in your workflow can deposit, pull, and supersede context. Claude, GPT, Gemini, custom agents — same protocol, shared whiteboard.

⚙ Vendor-Neutral

Works across LLM providers. No lock-in. Switch generators without losing context. Built on open formats and Postgres.

⇄ Human-in-the-Loop

Humans see the same context their agents do. Open questions surface. Handoff notes carry intent. Async collaboration that actually works.

◠ Immutable Journal + Mutable Truth

Packages are the historical record. Facts are the current truth. Wrong handoffs don't stay wrong forever — supersede, don't rewrite.

☀ Best-in-Class Retrieval

Hybrid search + cross-encoder reranking + gradient time-windowing. 100% recall@5 on LongMemEval Oracle, 97% on the harder S-variant.

☷ MCP-Native

Drop-in for Claude Code, Cursor, and any MCP-compatible agent. Your agents start oriented. Every handoff is a structured message.

§ Open Protocol Spec

Relay is the reference implementation of an open spec — the Agentic Protocol. Wire formats, conformance levels, and operations are public so other tools can interoperate. Read the v0.1 spec →
